# LED-Button-Power-PCB

 LED Button Power PCB (KiCad)

## Description

This project is a PCB design where an LED is controlled using a push button with external power input.

##  Components

* LED
* Resistor (1k)
* Push Button
* 2-Pin Connector

##  Learning

* PCB schematic design
* Footprint assignment
* Routing & layout
* Gerber generation
